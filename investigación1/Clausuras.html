Qué son las Clausuras?

Una clausura es una estructura de una función y su entorno léxico, incluidas las variables del alcance de la función en el momento de la creación de la clausura". Explicándolo de otra manera, con las clausuras en JavaScript se logra que una función interna tenga acceso al alcance de una función externa.

Para qué pueden servir?

Las clausuras son muy importantes en cualquier lenguaje de programación, para poder utilizar diferentes funciones y variables de manera externa

Cuando se ocupan?

Cuando se necesite enviar una función que tenga una varible interna que sea necesitada ejecutar antes de empezar el proceso

Ejemplo 1

function a(){
let x = 10;
function b(){
function c(){
function d(){
console.log(x);
}

}
c();
}
b();
}
a();
